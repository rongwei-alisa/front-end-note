# input pattern属性
`pattern` 属性规定用于验证输入字段的模式（正则），适用于 `type` 为`text`, `search`, `url`, `telephone`, `email` 以及 `password` 的 `input` 类型 

## invalid 伪类和 vaild 伪类
- valid 伪类，匹配通过 pattern 验证的元素
- invalid 伪类，匹配未通过 pattern 验证的元素
