# 数据类型
- 五种基本数据类型：Boolean, String, Number, Undefined, Null
- 一种复杂的数据类型：Object

## typeof 操作符
- 'undefined': 值未定义
- 'boolean': 值为布尔值
- 'string': 值为字符串
- 'number': 数值
- 'object': 值为对象或 null
- 'function': 值为函数
