# String

String 构造函数可以将其他类型的值转换成字符串
```
String(thing)
new String(thing)
```
## 字符串方法：
### 常用
* concat
* includes
* endsWith
* startsWith
* indexOf
* lastIndexOf
* match
* charAt
* padEnd
* padStart
* repeat
* replace
* search
* slice
* split
* substr
* substring
* toLowerCase
* toUpperCase
* toString
* trim trimStart trimEnd
* valueOf


### 不常用
* charCodeAt
* codePointAt
* localeCompare
* normalize
* toLocaleLowerCase
* toLocaleUpperCase
